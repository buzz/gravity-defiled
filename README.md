# Gravity Defiled

Gravity Defiled is a physics-based 2D motorcycle trial racing game. It is
inspired by the iconic game [Gravity
Defied](https://web.archive.org/web/20101228173717/http://www.codebrew.se/gdtr/)
from 2004.
